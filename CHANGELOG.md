# 变更日志

## [1.0.0] - 2026-03-22

### 新增 ✨

- **核心功能**
  - YP 平台商家数据采集
  - 追踪链接解析
  - 亚马逊商品爬取
  - 数据处理和清洗
  - 飞书文档同步

- **模块**
  - `YPMerchantCollector` - 商家采集器
  - `YPLinkParser` - 链接解析器
  - `AmazonCrawler` - 亚马逊爬虫
  - `DataProcessor` - 数据处理器
  - `FeishuClient` - 飞书客户端

- **特性**
  - 自动遵守 API 限流
  - 并发爬取支持
  - 错误重试机制
  - 数据去重
  - 多维度排序

- **文档**
  - README.md - 使用指南
  - QUICKSTART.md - 快速开始
  - DEPLOYMENT.md - 部署指南
  - ARCHITECTURE.md - 架构设计
  - PROJECT_OVERVIEW.md - 项目概览
  - CHANGELOG.md - 变更日志

### 技术栈 🛠️

- Python 3.9+
- Playwright (浏览器自动化)
- Requests (HTTP 客户端)
- BeautifulSoup4 (HTML 解析)
- Pandas (数据处理)
- 飞书 OpenAPI SDK

### 已知问题 ⚠️

1. YP API 可能需要登录认证（待验证）
2. 亚马逊反爬机制可能导致部分请求失败
3. 大规模采集可能触发限流
