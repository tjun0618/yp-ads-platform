# 工作日志 - 2026-03-22

## YP to Feishu 项目完成

### 项目概述

为用户开发了一个完整的自动化工具，用于从 YeahPromos (YP) 平台获取商家信息，爬取亚马逊商品数据，并同步到飞书文档。

### 完成的工作

#### 1. 需求分析和方案设计 ✅

**用户需求**：
- 获取 YP 平台上所有商家的商品信息
- 采集字段：商品名称、价格、描述、ASIN、评分、评论数、佣金率等
- 全量采集（所有商家、所有商品）
- 配置飞书应用并创建新的飞书应用

**技术方案**：
- 阶段 1：从 YP Monetization API 获取商家列表
- 阶段 2：解析 YP 追踪链接，提取亚马逊商品 URL
- 阶段 3：使用 Playwright 批量爬取亚马逊商品信息
- 阶段 4：数据清洗、去重、排序，同步到飞书表格

#### 2. 系统架构设计 ✅

创建了完整的架构文档 (`ARCHITECTURE.md`)，包括：
- 系统架构图
- 数据流程设计
- 技术栈选择
- 性能优化策略
- 反爬策略
- 错误处理机制

#### 3. 核心模块开发 ✅

**YP API 模块** (`src/yp_api/`):
- `merchant_collector.py` - 商家数据采集器
  - 支持分页查询
  - 自动遵守 API 限流（10次/分钟）
  - 自动重试机制

- `link_parser.py` - 链接解析器
  - 解析 YP 追踪链接
  - 提取亚马逊 URL
  - 提取 ASIN

**亚马逊爬虫模块** (`src/amazon/`):
- `crawler.py` - 商品爬虫
  - 使用 Playwright 自动化浏览器
  - 并发爬取（最多5个并发）
  - 提取商品名称、价格、评分、评论数等
  - 反爬策略

**数据处理模块** (`src/data/`):
- `processor.py` - 数据处理器
  - 数据清洗和标准化
  - 去重（基于 ASIN 和商品名称）
  - 多维度排序（评分、价格、佣金）
  - 过滤（价格范围、评分、库存）

**飞书集成模块** (`src/feishu/`):
- `client.py` - 飞书客户端
  - 认证管理（tenant_access_token）
  - 创建表格文档
  - 批量写入数据
  - 错误重试

**主程序** (`src/main.py`):
- 整合所有模块
- 实现 4 个阶段的完整流程
- 日志记录
- 错误处理
