# 🚀 快速开始 - 上传数据到飞书

## 📝 3步完成配置

### 第1步：创建飞书应用（2分钟）

1. 访问 https://open.feishu.cn/
2. 点击"创建应用" → "企业自建应用"
3. 应用名称填写：`YP数据上传`
4. 创建后进入应用详情页

### 第2步：获取凭证（1分钟）

1. 点击左侧"凭证与基础信息"
2. 复制 `App ID`（格式：cli_xxxxxxxxx）
3. 复制 `App Secret`（点击"查看"按钮）

### 第3步：配置脚本（1分钟）

打开文件 `quick_upload_to_feishu.py`，找到第24-25行：

```python
APP_ID = "cli_xxxxxxxxx"  # 🔴 请替换为您的飞书应用ID
APP_SECRET = "xxxxxxxxxxxx"  # 🔴 请替换为您的飞书应用密钥
```

替换为您的真实凭证，例如：

```python
APP_ID = "cli_1234567890abcdef"
APP_SECRET = "abcdefghijklmnopqrstuvwxyz123456"
```

---

## ✅ 开通权限

在飞书应用中：

1. 点击"权限管理" → "权限配置"
2. 搜索并开通以下权限：
   - `bitable:app` - 读取多维表格
   - `bitable:app:readonly` - 只读多维表格
   - `drive:drive` - 获取文件
3. 点击"批量申请权限"
4. 在飞书工作台中批准请求

## 🔄 再次上传（追加数据）

如果要将新数据追加到已有表格：

1. 修改 `quick_upload_to_feishu.py`
2. 在第28-29行取消注释并填入：

```python
# 使用现有表格
APP_TOKEN = "bascxxxxx"  # 替换为您的表格App Token
TABLE_ID = "tblxxxxx"  # 替换为您的表格ID
```

3. 保存并重新运行脚本

## ❓ 常见问题

### Q: 提示"权限不足"
**A**: 确保已在飞书应用中开通了所有必需权限，并在飞书工作台中批准了权限请求。

### Q: 提示"应用凭证无效"
**A**: 检查APP_ID和APP_SECRET是否正确复制，注意不要有多余的空格。

### Q: 上传到一半失败了
**A**: 可以直接重新运行脚本，已成功上传的数据不会重复。

### Q: 想要删除已上传的数据
**A**: 在飞书多维表格中手动删除，或使用飞书API删除记录。
