## 🐛 常见问题

### Q1: 登录失败怎么办？

**A**: 可能的原因：
- 验证码输入错误
- 用户名或密码错误
- 网络问题

**解决方法**：
- 重新运行脚本
- 确认凭证正确
- 检查网络连接

### Q2: 采集到的商家数据为空？

**A**: 可能的原因：
- Cookies 已过期
- API 端点不正确

**解决方法**：
- 重新登录获取新的 cookies
- 检查 YP 平台是否有更新

### Q3: 亚马逊爬取失败？

**A**: 可能的原因：
- 网络问题
- 亚马逊反爬限制
- 商品页面加载超时

**解决方法**：
- 增加请求延迟
- 使用代理
- 检查网络连接

### Q4: 飞书写入失败？

**A**: 可能的原因：
- 飞书应用配置错误
- App ID 或 Secret 不正确
- 权限不足

**解决方法**：
- 检查飞书配置文件
- 确认应用已发布
- 检查权限配置
