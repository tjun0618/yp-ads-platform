# YP to Feishu - 项目概览

## 🎯 项目简介

**YP to Feishu** 是一个自动化的数据采集和同步工具，用于从 YeahPromos (YP) 联盟营销平台获取商家信息，批量爬取亚马逊商品数据，并自动同步到飞书文档。

### 核心价值

- ✅ **自动化**：无需手动复制粘贴，全流程自动化
- ✅ **实时性**：按需采集，数据随时更新
- ✅ **结构化**：数据格式规范，便于分析
- ✅ **可扩展**：模块化设计，易于定制

### 模块说明

#### 1. YP API 模块 (`src/yp_api/`)

**功能**：与 YP 平台交互，获取商家数据

- `merchant_collector.py` - 商家数据采集器
  - 遵守 API 限流规则
  - 自动重试机制
  - 数据解析和清洗

- `link_parser.py` - 链接解析器
  - 解析 YP 追踪链接
  - 提取亚马逊 URL
  - 提取 ASIN

#### 2. 亚马逊爬虫模块 (`src/amazon/`)

**功能**：从亚马逊页面提取商品信息

- `crawler.py` - 商品爬虫
  - 使用 Playwright 自动化浏览器
  - 并发爬取支持
  - 反爬策略
  - 数据提取

#### 3. 数据处理模块 (`src/data/`)

**功能**：数据处理和转换

- `processor.py` - 数据处理器
  - 数据清洗
  - 去重
  - 排序和过滤
  - 统计分析

#### 4. 飞书集成模块 (`src/feishu/`)

**功能**：与飞书 API 交互

- `client.py` - 飞书客户端
  - 认证管理
  - 表格创建
  - 数据写入
  - 错误处理

## 🎯 核心功能

### 1. YP 商家数据采集

**功能**：从 YP Monetization API 获取所有商家信息

**数据字段**：
- 商家 ID
- 商家名称
- 佣金率
- 追踪链接
- 商家描述
- 商家类别

**特性**：
- 自动遵守 API 限流（10次/分钟）
- 自动重试机制
- 分页查询支持
- 数据验证

### 2. 链接解析

**功能**：解析 YP 追踪链接，提取亚马逊商品 URL

**解析逻辑**：
1. 识别 YP 追踪链接
2. 提取目标亚马逊 URL
3. 移除追踪参数
4. 提取 ASIN
5. 判断是否是商品页面

**支持格式**：
- 标准追踪链接：`https://yeahpromos.com/track?url=...`
- 短链接：`https://yp.to/abc123`
- 重定向链接

### 3. 亚马逊商品爬取

**功能**：批量爬取亚马逊商品信息

**数据字段**：
- ASIN（商品唯一标识）
- 商品名称
- 价格（含货币）
- 评分（1-5星）
- 评论数
- 商品描述
- 商品图片
- 品牌
- 类别
- 库存状态

**技术实现**：
- Playwright 浏览器自动化
- 动态内容渲染
- 并发爬取（最多5个并发）
- 自动延迟（2-3秒）
- 错误重试

### 4. 数据处理

**功能**：数据清洗、去重、排序

**处理步骤**：
1. 数据标准化
2. 去除重复商品
3. 数据验证
4. 排序（按评分/价格/佣金）
5. 统计分析

**过滤选项**：
- 价格范围过滤
- 评分过滤
- 库存状态过滤

### 5. 飞书同步

**功能**：自动创建飞书表格并写入数据

**同步内容**：
- 创建新表格文档
- 定义列结构（13列）
- 批量写入数据（500行/批）
- 自动分批处理

**表格结构**：
```
| 商家名称 | 商品ASIN | 商品名称 | 价格 | 佣金率 | 评分 | 评论数 | ... |
```
