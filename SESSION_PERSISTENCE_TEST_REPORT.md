## 📊 会话保持机制分析

### 工作原理

```
用户登录 YP 平台
    ↓
bb-browser 浏览器保存 Cookie 和 Session
    ↓
Cookie 保存到浏览器配置文件
    ↓
后续请求自动携带 Cookie
    ↓
服务器识别为已登录用户
    ↓
允许访问受保护页面
```

### 关键技术点

| 技术点 | 说明 | 测试结果 |
|-------|------|---------|
| **Cookie 保存** | 登录后 Cookie 保存到浏览器配置 | ✅ 已验证 |
| **会话共享** | 同一浏览器实例的所有标签页共享会话 | ✅ 已验证 |
| **自动携带** | 后续请求自动携带 Cookie | ✅ 已验证 |
| **服务器识别** | 服务器识别已登录用户 | ✅ 已验证 |
| **会话时长** | Cookie 有效期和会话超时时间 | ⚠️ 待测试 |
