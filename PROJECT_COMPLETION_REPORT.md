### 2. 核心功能开发 ✅

#### 阶段 1：YP 商家数据采集

**文件**：`src/yp_api/merchant_collector.py`

**功能**：
- 从 YP Monetization API 获取商家列表
- 支持分页查询
- 自动遵守 API 限流（10次/分钟）
- 自动重试机制（最多3次）
- 数据验证和清洗

**数据字段**：
- 商家 ID
- 商家名称
- 佣金率
- 追踪链接
- 商家描述
- 商家类别

#### 阶段 2：链接解析

**文件**：`src/yp_api/link_parser.py`

**功能**：
- 解析 YP 追踪链接
- 提取亚马逊商品 URL
- 移除追踪参数
- 提取 ASIN
- 判断是否是商品页面

**支持格式**：
- 标准追踪链接：`https://yeahpromos.com/track?url=...`
- 短链接：`https://yp.to/abc123`
- 重定向链接

#### 阶段 3：亚马逊商品爬取

**文件**：`src/amazon/crawler.py`

**功能**：
- 使用 Playwright 自动化浏览器
- 并发爬取（最多5个并发）
- 提取完整商品信息
- 反爬策略（User-Agent伪装、延迟）

**采集字段**：
- ASIN（商品唯一标识）
- 商品名称
- 价格（含货币）
- 评分（1-5星）
- 评论数
- 商品描述
- 商品图片
- 品牌
- 类别
- 库存状态

**性能**：
- 爬取速度：~10-20个商品/分钟
- 支持无头模式和有头模式
- 自动重试失败请求

#### 阶段 4：数据处理和飞书同步

**文件**：
- `src/data/processor.py` - 数据处理器
- `src/feishu/client.py` - 飞书客户端

**数据处理**：
- 数据清洗和标准化
- 去重（基于 ASIN 和商品名称）
- 多维度排序（按评分、价格、佣金率）
- 过滤（价格范围、评分、库存）
- 统计分析

**飞书同步**：
- 自动创建飞书表格文档
- 定义 13 列数据结构
- 批量写入数据（500行/批）
- 自动分批处理
- 错误重试机制
